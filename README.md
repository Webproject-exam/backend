
# Plantrol

This is the front-end of Plantrol.
URL: https://webproject-exam.netlify.app/

## Installation

 1. Download the code form GitHub. 
 2. Use the package manager
    [npm](https://www.npmjs.com/get-npm) to launch Plantrol.

```
npm install
```

## Usage

```
npm install

npm start
```
